# Cypress-Setup

To setup this repository follow below steps.

    1- Open New Terminal.
    Type one by one following commands. 
    2- npm init -y
    3- Type git clone https://github.com/ansabsajjad/Icon-Exchange.git
    4- npm --save -dev -y
    5- npm i

After running above commands your cypress is ready to use.
